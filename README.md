# mcenter
一个基于Django和Vue的个人博客。
> 网站地址：www.darkreunion.me


## 支持功能
### 文章分类
- 每篇博文均属于一个类别
- 可根据类别筛选文章

### 文章归档
- 根据每篇博文的发布时间进行存档
- 自动记录博文的创建时间
- 可根据存档时间来筛选博文

### 编写文章
- 以markdown格式编写文章
- 保存暂不发布的文章草稿
- 通过Django命令批量导入文章


## 项目结构
- backend文件夹: Django后端相关代码
- fontend文件夹: Vue前端相关代码
